Personvernerklæring

Vi bruker ikke informasjonskapsler eller analyseverktøy, og vi legger ikke inn sporingsskript fra tredjeparter. De eneste personopplysningene vi mottar direkte er det du velger å sende oss på e-post.

1. Behandlingsansvarlig

R&D Nordic er behandlingsansvarlig for personopplysningene som beskrives her. Kontakt: contact@rdnordic.com.

2. Hvilke personopplysninger vi behandler

  • E-postkorrespondanse: e-postadresse, navn hvis oppgitt, meldingsinnhold og eventuelle vedlegg.
  • Tekniske driftslogger: GitHub Pages kan behandle IP-adresse og relatert teknisk metadata for sikkerhet og misbruksforebygging.

3. Formål og behandlingsgrunnlag (GDPR art. 6)

Vi behandler personopplysninger for å svare på henvendelser, følge opp dialog før en mulig avtale, og opprettholde nødvendige kommunikasjonslogger.

  • Art. 6(1)(b): behandling som er nødvendig for å gjennomføre tiltak på din forespørsel før avtaleinngåelse.
  • Art. 6(1)(f): vår berettigede interesse i å svare på meldinger og dokumentere forretningskommunikasjon.

4. Databehandlere og overføringer

Vi bruker leverandører som behandler data på våre vegne:

  • Proton Mail for e-posttjenester.
  • GitHub Pages for hosting av nettstedet.

Noe behandling kan innebære overføring utenfor EØS/UK. Slike overføringer skjer med leverandørenes avtalte garantier (for eksempel standard personvernbestemmelser), slik dette beskrives i deres personverndokumentasjon.

5. Lagringstid

Vi lagrer henvendelser i inntil 12 måneder, og sletter deretter, med mindre lovpålagte krav krever lengre oppbevaring. Du kan be om tidligere sletting der det er aktuelt.

6. Dine rettigheter

Du kan be om innsyn, retting, sletting, begrensning, dataportabilitet og protestere mot behandling basert på berettiget interesse. Du kan også klage til Datatilsynet.

7. Kontakt om personvern

For spørsmål eller forespørsler om personvern, send e-post til contact@rdnordic.com. Vi kan be om tilstrekkelig informasjon for å bekrefte identitet før vi behandler en forespørsel.
